# Claxi Mobile (Starter)

This folder contains a minimal Expo-based React Native starter app.

## Why Expo

Expo is used for a fast, low-friction setup while mobile product flows are still being planned.

## Run locally

```bash
npm install
npm run start
```

Then choose one target from the Expo CLI:

- iOS simulator (`i`)
- Android emulator (`a`)
- Web preview (`w`)

## Current scope

- Starter screen only
- No Firebase wiring yet
- No web feature porting yet

This keeps mobile setup clean and ready for incremental development.
